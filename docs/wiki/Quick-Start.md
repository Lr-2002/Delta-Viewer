# 快速开始

## 1. 登录本地账号

首次启动时创建账号，之后使用同一账号登录。账号只用于记录“这条数据由谁处理”，不是云账号。密码没有服务器备份，也没有远程找回入口。

## 2. 选择 SD 卡

点击顶部的“选择 SD 卡”，选择操作系统已经挂载的卡根目录或本地测试目录。应用会立即扫描直接子目录中的全部 session，并在当前用户的 app-local-data/imports 工作区自动排队导入。不会再出现本地目标目录选择框。

源卡只读。应用不会在源卡上创建、重命名、修改或删除内容；本地工作区由应用自动创建。

## 3. 等待自动导入和检查

应用自动执行容量预检、文件复制、逐文件大小与 BLAKE3 回读校验。首个成功导入的 session 会继续做状态检查和 JPEG 抽检；进度条会显示阶段、当前路径、文件或字节进度以及耗时。

成功导入前只会出现带应用 marker 的 partial 目录；完整校验通过后才发布正式本地目录。已有同名目录不会被覆盖。

## 4. 选择其他 session

左侧列表只用于选择 session。单击更新选中项；双击，或聚焦后按 Enter/空格，才进入正常回放：

- 已经加载的 session 直接进入回放。
- 尚未加载的 session 已在后台自动导入时直接进入回放；导入失败的 session 双击或聚焦后按 Enter/空格可重试，错误会保存在顶栏的操作错误历史。

## 5. 检查、标注与导出

先查看[数据检查](Data-Validation)结果，再到回放页确认五路画面和状态曲线。保存任务、描述和轨迹码后，可以在导出页选择 MCAP、HDF5 或 LeRobot v2.1。
