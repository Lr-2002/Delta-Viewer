# 回放与裁剪

回放页在同一 `frame_id` 上显示五路相机画面，并同步位置、速度、欧拉角和角速度曲线。播放、暂停、前后步进、时间轴和 `0.25x / 0.5x / 1x / 2x` 速度均作用于同一帧位置。

无法读取的当前帧会显示明确错误，不会继续显示上一帧冒充当前帧。点击带帧号的数据检查 issue 可以跳转到对应位置。

## 选择轨迹范围

轨迹范围是一个连续的闭区间，起点和终点都包含在导出中。可以使用双范围滑块、数字输入，或把当前帧设为起点/终点。重置后恢复完整轨迹。

裁剪只改变当前回放与本次导出：

- 不修改源 SD 卡。
- 不修改本地导入副本。
- 不重新复制或重编码范围外 JPEG。
- MCAP、HDF5 和 LeRobot 使用同一范围。

范围外、能够绑定到具体帧的 issue 不阻断本次导出；范围内 issue 和无法绑定帧的全局 issue 仍按错误/警告规则处理。
